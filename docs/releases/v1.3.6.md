# Taurus V1.3.6 Release Notes

## 中文

Taurus V1.3.6 完成了 P1 计算核心重构，并修复了 ARRI 摄影机分辨率选择回归问题。

### 主要变化

- 引入可独立测试的 `CameraSelection`、`RecordingMode`、`MediaSpec`、`RecordingMetrics`、`CalculationResult` 和 `CalculationEngine` 稳定类型。
- 将计算逻辑从 SwiftUI 视图中分离，并统一不同品牌的结果展示。
- 根据 ARRI FDRC v2.6.3 官方数据快照使用显式录制模式和介质限制表。
- 修复 ARRI 所有官方摄影机与编码组合的分辨率选择；同时增加其他品牌的代表性回归测试。
- 保持 macOS 12 最低版本、离线使用和零第三方运行时依赖。

## English

Taurus V1.3.6 completes the P1 calculation-core refactor and fixes the ARRI resolution-selection regression.

### Highlights

- Added independently testable `CameraSelection`, `RecordingMode`, `MediaSpec`, `RecordingMetrics`, `CalculationResult`, and `CalculationEngine` types.
- Separated calculation logic from SwiftUI views and unified result presentation across brands.
- Switched ARRI to explicit recording modes and media-constraint tables based on the official ARRI FDRC v2.6.3 data snapshot.
- Fixed resolution selection for all official ARRI camera and codec combinations, with representative regression coverage for other brands.
- Maintains macOS 12 as the minimum version, offline operation, and zero third-party runtime dependencies.

### Release Information

- Display name: `Taurus V1.3.6`
- Version: `1.3`
- Build: `6`
- Release tag: `v1.3.6`
