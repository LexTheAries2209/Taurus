# Taurus V2.0.1 Release Notes

## 中文

Taurus V2.0.1 将应用从单一数据计算器拓展为离线 macOS DIT/DMT 工作工具。新版本保留原有摄影机数据计算流程，并加入可保存、比较和导出的多机位 DMT 项目规划。

### DMT 项目规划

- 通过顶部标签在数据计算器和 DMT 项目规划之间切换。
- 为每个机位设置摄影机录制模式、介质、机位名称与备注、摄影机数量、每日开机时长、实际录制比例、拍摄天数、保留副本和安全余量。
- 为每个机位独立设置读卡器速度，并设置项目目标磁盘速度和每日卸载窗口。
- 汇总单机每日、全部机位每日、全项目原始数据量和备份后存储需求。
- 计算介质周转卡次、单张介质录制时长、机位卸载时间、全项目卸载时间以及每日双备份可行性。
- 为支持的 ARRI ARRIRAW 模式提供 HDE 无损压缩开关，包括 AMIRA。
- 支持摄影机搜索、收藏模式、2 至 4 个模式并排比较、机位复制、重排、重置与删除。
- 支持项目复制、Codable JSON 自动保存和 JSON 导入导出。
- 支持 CSV 与 PDF 项目报告，包含项目参数、容量、介质周转、备份和传输时间。

### 数据计算器与界面

- 保留离线摄影机、编码、分辨率、帧率和介质查询及容量计算。
- 级联选择会立即清理失效的下游值；无效或不完整组合不会输出数字。
- `[General]` 手动输入仅接受大于零的有限数字。
- 数据计算器与 DMT 项目规划均支持中英文界面。
- 改进默认窗口尺寸、顶部工具栏、表格列宽、Inspector 输入对齐和工作区居中缩放动画。
- 修复多次切换工作区后窗口位置累计漂移的问题。

### 本地数据与兼容性

- 项目和收藏保存在用户 Application Support 目录，不依赖账户或云服务。
- 旧 `DITProjects.json` 与 `DITFavorites.json` 会自动迁移到 DMT 文件名。
- 保持 macOS 12 最低版本、离线核心功能、零第三方依赖和 `GPL-3.0-only` 许可。

## English

Taurus V2.0.1 expands the app from a standalone data calculator into an offline macOS DIT/DMT workflow tool. It preserves the existing camera-data calculation workflow and adds saved, comparable, exportable multi-camera DMT project planning.

### DMT Project Planner

- Switch between Data Calculator and DMT Project Planner from the top workspace control.
- Configure each camera position with a recording mode, media, name and notes, camera count, daily power-on time, actual recording ratio, shoot days, retained copies, and safety margin.
- Set reader speed per camera position, plus project-level target-disk speed and daily offload window.
- Summarize raw data per camera per day, all cameras per day, the full project, and storage after backups.
- Calculate media turnover cycles, recording time per card, per-camera offload time, total offload time, and daily double-backup feasibility.
- Enable HDE lossless compression for supported ARRI ARRIRAW modes, including AMIRA.
- Search cameras, save favorite modes, compare 2 to 4 modes, and duplicate, reorder, reset, or delete camera positions.
- Duplicate projects, autosave Codable JSON, and import or export project JSON.
- Export CSV and PDF project reports containing parameters, capacity, media turnover, backups, and transfer timing.

### Data Calculator and Interface

- Preserves offline camera, codec, resolution, frame-rate, media lookup, and capacity calculation.
- Cascading selection immediately clears invalid downstream values; invalid or incomplete combinations do not produce numeric results.
- `[General]` manual inputs accept only finite numbers greater than zero.
- Both Data Calculator and DMT Project Planner support Chinese and English interfaces.
- Refines default window sizing, the top toolbar, resizable table columns, Inspector input alignment, and centered workspace scaling.
- Fixes cumulative window-position drift after repeated workspace switching.

### Local Data and Compatibility

- Projects and favorites remain in the user's Application Support directory without accounts or cloud services.
- Legacy `DITProjects.json` and `DITFavorites.json` files migrate automatically to the DMT filenames.
- Retains macOS 12 support, offline core behavior, zero third-party dependencies, and the `GPL-3.0-only` license.
