# Taurus V1.3.7 Release Notes

## 中文

Taurus V1.3.7 是一次 ARRI 兼容性修复版本，恢复旧版选择器的显示与选项范围，同时保留当前官方计算数据。

### 主要变化

- 恢复 ARRI 分辨率的旧版显示格式，并将界面显示名称与官方计算模式分离。
- 保留官方数值：ALEXA XT 3.2K 为 `3164*1778`，ALEXA 265 5.1K 为 `5120*3100`。
- 使用已确认的旧版格式显示 ALEXA Mini ARRIRAW 1.9K/2.5K 和 ALEXA 35 系列 3.8K 变形模式。
- 不加入 AMIRA MPEG-2 HD 422；恢复 `ALEXA 35 Live Xtreme` 机型入口。
- 恢复旧版 ARRI 帧率候选数量；官方介质上下限仍决定最终可选范围。

## English

Taurus V1.3.7 is an ARRI compatibility release that restores the legacy selector presentation and option range while retaining the current official calculation data.

### Highlights

- Restored the legacy ARRI resolution display format and separated UI display names from official calculation modes.
- Preserved official dimensions: ALEXA XT 3.2K is `3164*1778`, and ALEXA 265 5.1K is `5120*3100`.
- Applied the confirmed legacy-style names for ALEXA Mini ARRIRAW 1.9K/2.5K and the ALEXA 35-series 3.8K anamorphic mode.
- Did not add AMIRA MPEG-2 HD 422, and restored the `ALEXA 35 Live Xtreme` camera entry.
- Restored the legacy ARRI frame-rate choice set; official media limits still determine the final available range.

### Release Information

- Display name: `Taurus V1.3.7`
- Version: `1.3`
- Build: `7`
- Release tag: `v1.3.7`
- Download: `Taurus-v1.3.7-macOS.zip`
- Architectures: arm64 and x86_64 universal binary
- ZIP size: 3,473,936 bytes
- SHA-256: `d4e33b55cefdc2ad242df1547755da5df00221627f15bb68bc9f55bda0e4477f`
- Signing: ad-hoc signed; `codesign --verify --deep --strict` passed
- Notarization and Gatekeeper: not notarized; local `spctl` assessment was rejected, so first launch may require confirmation in System Settings

### Verification

- Full XCTest: passed.
- Release build: passed.
- Independent build from the archived source: passed.
- Swift 6 compilation check: completed with the existing `PickerView` AppKit main-actor warnings.
- Strict concurrency with warnings treated as errors: not passed because of the existing `PickerView` AppKit main-actor isolation warnings.

The calculations are intended for on-set planning and quick reference. Actual data rates may vary with firmware, camera settings, variable frame rates, compression behavior, external recording workflows, and usable media capacity.
