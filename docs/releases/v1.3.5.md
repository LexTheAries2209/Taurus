# Taurus V1.3.5 Release Notes

## 中文

Taurus V1.3.5 是一次正式功能更新，重点补充英文界面、截图导出、ARRI ALEXA 35 Live Xtreme 数据支持，并整理应用内注释与仓库文档。

### 更新内容

- 加入英文界面翻译，数据计算器的主要界面文本现在支持英文显示。
- 加入右上角截图功能，便于导出当前计算界面。
- 加入 ARRI ALEXA 35 Live Xtreme 机型的选择和计算。
- 更新底部注释，将特别鸣谢移入 README，并在应用内介绍 GitHub 仓库。
- 调整版本命名规则：Version 保持为 `1.3`，Build 为 `5`，应用显示为 `Taurus V1.3.5`。

### 说明

- 摄影机规格参数数据截止至 2026-07-06。
- 计算结果仅供现场规划和快速参考，实际数据量可能受到厂商固件、机内设置、可变帧率、压缩策略、外录流程和记录介质实际可用容量影响。

## English

Taurus V1.3.5 is a formal feature release focused on English UI support, screenshot export, ARRI ALEXA 35 Live Xtreme data support, and refreshed in-app notes and repository documentation.

### Changes

- Added English UI translation for the main data-calculator interface.
- Added a screenshot button in the upper-right corner for exporting the current calculator view.
- Added selection and calculation support for ARRI ALEXA 35 Live Xtreme.
- Updated the in-app notes: acknowledgements moved to the README, and the app now points users to the GitHub repository.
- Updated version naming: Version stays at `1.3`, Build is `5`, and the app displays as `Taurus V1.3.5`.

### Notes

- Camera specification data is current through 2026-07-06.
- Calculated results are intended for planning and quick reference. Real-world data usage can vary because of firmware versions, in-camera settings, variable frame rates, compression behavior, external recording workflows, and actual usable media capacity.
